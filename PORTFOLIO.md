# LLM 기반 뉴스 데일리 브리핑 자동화 파이프라인

## 포트폴리오 개요

외부 수집 데이터로부터 매일 뉴스 브리핑 콘텐츠를 생성하는 Python 자동화 파이프라인 데모입니다. 뉴스 분류, 선별, 요약, 시사점 작성, HTML 생성, 음성 스크립트 생성을 단계별로 처리하는 구조를 보여줍니다.

이 데모는 실제 API 키 없이 mock 로직으로 동작하며, 위시켓 포트폴리오 제출을 위한 유사 구조 구현 예시입니다.

## 작업 범위

- 단계별 JSON 입출력 구조 설계
- 뉴스 사전 필터링
- 우선순위 정렬 및 최종 수록 대상 선정
- 요약/시사점 생성 mock 로직
- HTML 브리핑 출력
- 체크포인트 저장 구조

## 주요 기능

- 원본 뉴스 JSON 로딩
- 필수 필드 기준 사전 필터링
- 점수 기반 우선순위 정렬
- 요약 및 시사점 생성 단계 분리
- `output/briefing.html` 결과물 생성

## 기술적 주안점

LLM을 한 번만 호출해 전체 결과를 생성하는 구조는 결과 품질과 디버깅 측면에서 한계가 있습니다. 이 데모는 뉴스 데이터 처리 과정을 여러 단계로 나누고, 각 단계의 입력과 출력을 JSON으로 관리해 안정성과 재현성을 높이는 구조를 기준으로 했습니다.

수록 건수, 정렬, 데이터 검증처럼 재현성이 필요한 부분은 코드로 처리하고, 요약과 시사점 작성처럼 생성 품질이 중요한 부분은 LLM 호출 단계로 분리하는 방향을 가정했습니다.
